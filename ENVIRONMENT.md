# 🌱 Environment Setup

## 🔧 Backend

- Node.js v18+
- Express.js
- MongoDB (Atlas or local)
- JWT for authentication

## 🖥️ Frontend

- React.js
- Axios for HTTP requests
- Bootstrap / Custom CSS

## 📦 Dev Tools

- VS Code
- Git & GitHub
- Postman for API testing

## 💻 Hardware

- Min: 8GB RAM, 25GB free disk space
